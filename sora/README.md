# sora

这个目录存放 Sora 相关的视频生成脚本与工具。

常用文件：
- `test.py`：基础视频生成客户端
- `generate_video_with_character.py`：带角色的视频生成脚本
- `create_sora_character.py`：创建 Sora 角色
- `submit_jingang_tasks.py` / `wait_jingang_tasks.py`：批量提交与轮询《金刚经》任务
- `submit_tianting_tasks.py` / `resubmit_failed_tianting.py`：天庭题材的 Sora 任务脚本

建议在本目录下执行相关命令。
